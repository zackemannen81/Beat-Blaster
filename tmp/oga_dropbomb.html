I made some tests using this pack with the following tools:

Flash

Game Maker 8.0 Pro Edition

Dark Basic

Multimedia Fusion 2

Vale Hammer Editor

 

These are the results:

 

Flash: totally compatible;

Game Maker 8.0 Pro Edition: for some unknown reason, when the program runs one of these sounds, the game suffers a delay and it is paused for a fraction of second. I didn't tested on Game Maker Studio yet;

 

Dark Basic: totally compatible;

 

Multimedia Fusion 2: totally compatible;

 

Valve Hammer Editor: this engine only supports wav files. His similar, Jack Hammer, too.
